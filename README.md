# Level Bet

**Удобный трекер ставок с подробной статистикой и аналитикой результатов**
- добавление и редактирование купонов
- учет событий по видам спорта и их коэффициентов
- фильтры по периоду и статусу
- расчет профита, ROI, винрейта и других показателей
- детальная аналитика для ROI и винрейта
- любопытные метрики и игровые серии
- встроенный глоссарий показателей
- защита входа через Face ID

Приложение помогает замечать закономерности, следить за дисциплиной и лучше понимать собственные игровые привычки.
Предназначено только для ведения статистики и анализа результатов.

**Платформа:** SwiftUI

**Архитектура:** MV

**Работа с данными:** SwiftData

**Авторизация:** LocalAuthentication (Face ID)

**Доп. фреймворки:** Charts
